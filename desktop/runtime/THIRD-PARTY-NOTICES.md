# Desktop runtime notices

Omplish ships target-specific Node.js and PostgreSQL runtime artifacts in the desktop installer.
The exact archive URLs, versions, and SHA-256 values are recorded in `manifest.json`.

## Node.js

Node.js is distributed under the MIT license. The upstream project and complete license/source
notices are available at <https://github.com/nodejs/node> and in the corresponding pinned Node.js
release archive.

## PostgreSQL

PostgreSQL is distributed under the PostgreSQL License. The upstream project and license text are
available at <https://www.postgresql.org/about/licence/> and in the corresponding pinned PostgreSQL
binary archive.

The release build record must retain the original archives, their SHA-256 values, and any notices
included by the upstream distributions.
